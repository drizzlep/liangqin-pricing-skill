# Public Release Kit

这个目录是给 `liangqin-contract-pricing-audit` 新 GitHub 仓库准备的公开发布骨架包。

用途：

- 先在当前仓库里把公开发布需要的标准文件整理齐
- 后续新建 GitHub 仓库时，直接把 `liangqin-contract-pricing-audit/` 整体拷过去
- 再补最终代码、脱敏样例和仓库地址

注意：

- 这里是发布模板，不是当前运行环境入口
- 不包含真实客户合同、敏感价格数据或内部私有素材
- `liangqin-contract-pricing-audit` 依赖 `liangqin-pricing`，不能脱离后者独立运行
