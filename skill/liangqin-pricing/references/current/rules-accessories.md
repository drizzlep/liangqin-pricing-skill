# 配件类规则

适用品类：

- 轨道插座
- 岩板台面、岩板背板、铝框岩板门板
- 金属洞洞板
- 铜套脚
- 箱体床举升器
- 特殊木拉手
- 抽屉收纳格
- 按弹抽屉
- 灯带分区开关
- 脚轮
- 穿衣镜

## 轨道插座

- 电力轨道按米或固定规格计价
- 活动插座按个计价
- 需区分 T25 和西蒙体系

## 岩板

- 岩板台面：1460 * 岩板长度 + 柜体正常计算
- 铝框岩板门板：1860 * 岩板长度 + 无门柜体价格
- 岩板背板：1460 * 岩板长度
- 空区高度小于 55cm 时不计算侧板
- 空区高度大于等于 55cm 时，超出侧板面积 * 材质单价
- 岩板台面、铝框岩板门板缺关键尺寸时，先问 `岩板长度`
- 岩板背板先问 `空区高度`
- 如果岩板背板空区高度大于等于 55cm，再问 `超出侧板面积`
- 不自动从整体宽高深反推 `超出侧板面积`

### 岩板正式报价写法

- 正式报价必须拆开写 `基础柜体价格`、`岩板加价`、`侧板加价（如有）`、`最终小计`
- 岩板台面、铝框岩板门板：
  - 如果基础柜体金额已齐，直接给正式报价
  - 不把岩板加价混进基础单价
- 岩板背板：
  - `空区高度 < 55cm` 时，侧板加价写 `0`
  - `空区高度 ≥ 55cm` 且 `超出侧板面积` 已齐时，再把侧板加价展开
- 如果当前是在问规则或缺参：
  - 台面 / 铝框岩板门板先只问 `岩板长度`
  - 背板先只问 `空区高度`
  - 满足 `空区高度 ≥ 55cm` 后，再问 `超出侧板面积`

## 箱体床举升器

- 这一节是 `配件升级加价` 规则，不是床类默认补问规则
- 如果用户当前只是问 `床垫重量未知还缺什么`、`举升器怎么按规则判断`
  - 先按床类主规则回答 `床垫重量应≤50kg / 超重可两套750N / W＞1800默认两套750N需单独收费 / 下单需备注`
  - 在 `床垫重量未知` 的这一轮，不要直接写 `一套750N` 结论，也不要整理成条件对照表
  - 不先展开本节的升级加价项
- 只有在用户进一步追问 `要不要改电动举升器`、`升级举升器加多少钱` 时，才使用本节
- 电动举升器在原价基础上加 1000 元
- 小蜻蜓举升器在宽度大于 1800 时默认两套，加 600 元
