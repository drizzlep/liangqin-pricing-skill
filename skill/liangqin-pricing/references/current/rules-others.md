# 其他单品规则

适用品类：

- 床头柜
- 几案 / 茶几
- 斗柜
- 沙发
- 悬空挂墙层板
- 标品电视柜加抽屉
- 其他目录中按单件计价的标准品

## 总体原则

- 这一类产品优先按产品目录标准品价格计价
- 小改一般考虑加价 15%
- 大改按比例计算或按最接近标准尺寸高档靠齐
- 如同一逻辑下两种算法都有道理，取较高值

## 床头柜 / 几案 / 斗柜

- 优先按目录中对应型号、尺寸、材质单件价报价
- 若只是少量尺寸改动，优先参考标准品价并结合比例或 15% 调整

## 标品电视柜加抽屉

- 多加抽屉按个收费

## 悬空挂墙层板

- 按米计价
- 不区分进深，按长度计算

## 沙发

- 标准沙发按单人、双人、三人查表
- 长滩沙发与长滩沙发 Y 使用独立价格表
- 罗汉床 / 榻按同尺寸沙发 1 价格增加 30%
- 新款沙发垫按单人、双人、三人另加
