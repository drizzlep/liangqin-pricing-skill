# 床类规则

适用品类：

- 架式床
- 箱体床
- 儿童床
- 榻榻米
- 抽屉床
- 榻榻米加超大抽屉

## 必问字段

- 床类别
- 床垫宽度和长度
- 材质
- 是否标准品还是非标
- 是否加高、换板、加宽床头、增加抽屉或举升器

## 通用规则

### 尺寸口径

- 床垫宽度 1.2m 及以下，同 1.2m 价格
- 宽度 1.35m 以上按 1.5m 算
- 床垫长度 1.9m 也按 2m 算

### 标品无 1.2m 价格

- 1.2m 价格 = 1.5m 价格 - (1.8m 价格 - 1.5m 价格)

### 超大床

- 以该床 1.5m 宽价格为基础按比例计算

### 加高

- 架式床加高统一按整床价格 +15%

## 部件和改动

- 排骨架换平板拼框床板按材质加价
- 无床头架式床 / 箱体床（塌式床）按材质减价
- 床头板加宽按加宽尺寸、床宽、材质查表加价
- 只有用户明确提到这些部件或改动时，才进入对应加价或减价路径
- 如果用户没有提到举升器、气压杆、抽屉、换板、床头加宽，不主动补默认配置说明

## 儿童床

- 非标儿童床按非标儿童床报价方案执行
- 儿童高架床按挂梯或梯柜加价表处理

### 模块化儿童床报价

- 明确标准成品且能命中目录儿童床记录时，继续按目录成品价报价。
- 明确 `定制 / 非标`，或用户说的是 `半高床 / 高架床 / 错层床`，或标准目录无法命中时，转入模块化儿童床报价。
- 模块化儿童床当前独立于主 `price-index.json`，按模块组合计价，不和目录成品价混算。
- 模块化儿童床固定补问顺序：
  - `床形态`
  - `上层出入方式`
  - `床垫宽度 / 长度`
  - `下层结构（上下床 / 错层床时）`
  - `材质`
  - `围栏样式`
  - `围栏尺寸（若该组合不是固定特价模块）`
  - `梯子 / 梯柜尺寸（若该组合不是固定特价模块）`
- 半高床统一按 `高架床单层模块组合` 处理，不单独拆第三套算法。
- 半高床 / 高架床床下如果带柜体，按 `床体模块价 + 床下柜体投影面积价` 分开计算后合并。
- 床下前后双排柜体当前要求前排、后排 `进深分别确认`，不允许只给一个总进深直接正式报价。
- 床下双排柜体第一阶段只支持 `单排进深 ≤ 450mm` 的组合报价；超过时先返回限制说明。
- 床下组合柜路径当前如果缺 `后排进深`，不能用 `前排进深` 自动代替，也不能先按互通默认同深正式报价。
- 床下组合柜路径当前如果 `单排进深 > 450mm`，不能拆成“床体模块价 + 常规定制衣柜”绕开限制继续正式报价。
- 当前已支持的床下柜体换算口径：
  - `有门无背板 = 无门无背板书柜基础单价 + 带门无背板门板单价`
  - `无门有背板 = 无门无背板书柜基础价 + 背板外露 10%`
- 模块化儿童床当前支持的组合模块包括：
  - 床架：`架式床 / 箱体床 / 高架床 / 大/小树屋`
  - 围栏：常规围栏、特殊围栏、城堡围栏
  - 出入件：`直梯 / 斜梯 / 梯柜`
  - 附加件：`下框板 / 拖抽 / 腿称`
- `乌拉圭玫瑰木` 命中特价模块时，优先按特价模块报价，不再叠加通用面积或长度公式。
- 上铺或高架床床垫宽度大于 `1.2m` 时，不直接正式报价，先按儿童床限制说明处理。
- 如果用户说的是 `经典护栏款` 这类非标准名称，必须先映射到标准围栏名称；在床下双排柜体场景里，如果同时缺前后柜体进深，可以同一轮一起追问。

## 榻榻米

- 按投影面积计价
- 单价由材质和高度区间决定

## 抽屉床

- 对应尺寸箱体床价格 + 抽屉个数 * 抽屉单价

## 榻榻米 + 超大抽屉

- 榻榻米投影面积价格 + 抽屉个数 * 抽屉单价
