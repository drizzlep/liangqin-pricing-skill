# 部件类规则

适用品类：

- 抽屉
- 挡边条
- 灯带
- 书梯
- 门板
- 背板
- 柜内茶格
- 拱形玄关挡板
- 圆弧侧板

## 抽屉 / 挡边条 / 灯带

- 抽屉按个计价
- 柜体通常每延米赠送 2 个抽屉，超出部分按个加钱
- 书柜抽屉是否算入有门或无门部分，按书柜结构判断
- 挡边条优先随柜体单价按投影面积计价；表格单价只作指导价
- 灯带按米计价

## 书梯

- 木质梯按米计价：总长 * 单价
- 轨道按米计价：长度 * 单价
- 需区分常规轨道和可回收轨道

## 门板

- 单独门板单价主要用于特殊柜体计算，不作为常规单卖报价依据
- 单独门板按投影面积单价计
- 吊轨门价格 = 门板单价 * 门板面积 + 400
- 屏风/樟纸门投影面积不足 1.6㎡ 时按 1.6㎡ 算

## 背板

- 非整体小面积外露空区，通常按专门空区规则处理
- 常规结构背板外露时，柜体总价加 10%

## 柜内茶格

- 按体积 * 单价计算
- 体积 = a * b * c

## 拱形玄关挡板

- 挡板弧形区域按平板门柜体价格计算
- 其余部分按正常柜体面积计算

## 圆弧侧板

- 按米计价
- 当前规则：300 元/米
